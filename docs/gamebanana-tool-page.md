## Body (WYSIWYG)

FunkHub is a desktop launcher and manager for Friday Night Funkin' mods.

Main features:
- Discover mods and categories from GameBanana
- Install mods into supported engine mod folders
- Install standalone executable mod packages
- Manage multiple engine instances
- Launch engines and standalone packages from one app
- One-click installs with `funkhub://`

Supported engines:
- Psych Engine
- Base Game / V-Slice
- Codename Engine
- FPS Plus
- JS Engine
- ALE Psych
- P-Slice

## Installation Instructions

1. Open releases: https://github.com/Crew-Awesome/FunkHub/releases
2. Download your platform build:
   - Windows: `FunkHub Setup <version>.exe` (recommended) or `FunkHub <version>.exe` (portable)
   - Linux: `.AppImage` or `.deb`
   - macOS: `.dmg` or `.zip`
3. Install/run FunkHub for your platform.
4. Launch FunkHub and configure your game/data folders.
5. Install an engine (if needed), then install mods from Discover or one-click URLs.

## Requirements

- Windows/Linux/macOS desktop environment
- Friday Night Funkin' files or compatible engine install paths
- Internet connection for GameBanana browsing/downloading
- Disk space for downloads, extracted archives, and engine installs
