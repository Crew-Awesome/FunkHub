# FunkHub v0.4.0 - Category Comeback

## Release Name
Category Comeback

## Summary
FunkHub v0.4.0 restores the full Discover category experience and improves reliability across mod installs, engine launching, and updater behavior.

## Highlights
- Restored Discover category browsing on desktop and mobile (sidebar + sheet) with search and tree expansion.
- Fixed provider-side category filtering so selected categories correctly affect Discover results.
- Restored stable Discover pagination resets when category/sort/search changes.
- Improved executable and archive install handling in desktop runtime.
- Improved engine launch behavior and added playtime tracking.
- Continued translation sync and UI quality updates.

## Notable Technical Changes
- Discover category UI and filtering behavior restored and wired end-to-end.
- Runtime install fallback behavior for tricky package shapes was improved.
- Update and launch flows received additional robustness fixes.
